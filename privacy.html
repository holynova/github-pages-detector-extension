Privacy Policy

GitHub Pages Detector checks whether the GitHub repository page you are viewing has GitHub Pages enabled.

English

The extension reads the current GitHub repository URL in order to determine the repository owner and name. It calls the GitHub Pages API and may request the inferred GitHub Pages URL to verify whether the page exists.

If you choose to save a GitHub token for private repository checks, the token is stored only in Chrome sync storage and is sent only to GitHub API endpoints needed for the extension's single purpose. The extension does not sell user data, serve ads, track browsing for advertising, or transfer user data to unrelated third parties.

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

中文

GitHub Pages Detector 会读取当前 GitHub 仓库页面的 URL，用于识别仓库 owner 和 repo 名称。插件会请求 GitHub Pages API，并可能访问推导出的 GitHub Pages URL，以判断 Pages 是否存在。

如果你选择保存 GitHub token 用于检测私有仓库，该 token 只保存在 Chrome sync storage 中，并且只会发送到实现插件单一功能所需的 GitHub API。插件不会出售用户数据，不展示广告，不进行广告追踪，也不会把用户数据转移给无关第三方。

从 Google API 获得的信息使用将遵守 Chrome Web Store User Data Policy，包括 Limited Use 要求。
