# Chrome Web Store Listing - English

## Name
GitHub Pages Detector

## Short Description
Detect GitHub Pages on GitHub repository pages and open or copy the Pages URL.

## Detailed Description
GitHub Pages Detector adds a clear GitHub Pages status directly to GitHub repository pages.

When you open a repository, the extension checks whether GitHub Pages is configured. It shows an inline status while checking, when Pages is available, and when Pages is not found. If Pages exists, you can open the site in a new tab or copy the detected Pages URL with one click.

Features:
- Detects GitHub Pages from GitHub repository pages
- Uses the GitHub Pages API first, then verifies the inferred Pages URL
- Shows checking, found, not found, and error states inline
- Opens the GitHub Pages URL in a new tab
- Copies the GitHub Pages URL with one click
- Optional GitHub token support for private repositories

The extension is focused on one purpose: helping developers quickly discover and open a repository's GitHub Pages site.
