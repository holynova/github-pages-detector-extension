# Chrome 应用商店上架文案 - 中文

## 名称
GitHub Pages Detector

## 简短描述
在 GitHub 仓库页面检测 GitHub Pages，并一键打开或复制 Pages 地址。

## 详细描述
GitHub Pages Detector 会在 GitHub 仓库页面中直接显示 GitHub Pages 状态。

当你打开一个仓库时，插件会检测该仓库是否配置了 GitHub Pages。页面中会显示检测中、已发现、未发现和检测失败等状态。如果仓库存在 Pages，你可以一键在新标签页打开，也可以一键复制检测到的 Pages URL。

功能：
- 在 GitHub repo 页面检测 GitHub Pages
- 优先使用 GitHub Pages API，再验证推导出的 Pages URL
- 在页面内显示检测中、已发现、未发现、失败状态
- 一键新标签页打开 GitHub Pages
- 一键复制 GitHub Pages URL
- 支持可选 GitHub token，用于私有仓库检测

插件只有一个目标：帮助开发者快速发现并打开 GitHub 仓库对应的 Pages 站点。
